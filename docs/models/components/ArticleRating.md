# ArticleRating

This schema contains general information about the article rating.


## Fields

| Field                                                                                                               | Type                                                                                                                | Required                                                                                                            | Description                                                                                                         |
| ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `id`                                                                                                                | *Optional\<String>*                                                                                                 | :heavy_minus_sign:                                                                                                  | The ID of the rating.                                                                                               |
| `articleId`                                                                                                         | *Optional\<String>*                                                                                                 | :heavy_minus_sign:                                                                                                  | The ID of the article.<br><br>An article ID is composed of a 2-4 letter prefix, followed by a dash and 4-15 digits. |
